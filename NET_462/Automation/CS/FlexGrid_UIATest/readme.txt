FlexGrid_UIATest
----------------------------------------
This project demonstrates how to UI test the C1FlexGrid control using Appium.Net and Windows Application Driver.

## Setup Instructions

1. Build the FlexGrid_UIATest project.
2. Run the application without debugging (e.g., `Ctrl + F5` in Visual Studio).
3. Ensure Windows Application Driver is installed.  
   After installing, run WinAppDriver.exe.
4. (Optional) To verify the WebDriver service is running, visit:  
   [http://127.0.0.1:4723/status](http://127.0.0.1:4723/status)  
   You should see a JSON response if it's working.
5. From your IDE, navigate to Test → Run → All Tests in Solution to execute the UI tests.
