## FinancialChartExplorer
#### [Download as zip](https://grapecity.github.io/DownGit/#/home?url=https://github.com/GrapeCity/ComponentOne-WPF-Samples/tree/master/NET_10/Chart/FinancialChartExplorer)
____
#### Showcases the WPF FinancialChart.
____
This sample contains pages that demonstrate financial specific chart types
and main features of WPF FinancialChart.

The sample includes the following pages:


* Chart Types
* Heikin-Ashi
* Line Break
* Renko
* Kagi
* ColumnVolume
* EquiVolume
* CandleVolume
* Arms CandleVolume
* Point&Figure


* Interaction
* Markers
* Range Selector


* Analytics
* Trend Lines
* Moving Average
* Overlays
* Indicators
* Event Annotations
* Fibonacci Tools
