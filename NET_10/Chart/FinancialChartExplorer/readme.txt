FinancialChartExplorer
------------------------------------------------------------------------------
Showcases the WPF FinancialChart.

This sample contains pages that demonstrate financial specific chart types
and main features of WPF FinancialChart.

The sample includes the following pages:

 * Chart Types
  - Heikin-Ashi
  - Line Break
  - Renko
  - Kagi
  - ColumnVolume
  - EquiVolume
  - CandleVolume
  - Arms CandleVolume
  - Point&Figure

 * Interaction
  - Markers
  - Range Selector

 * Analytics
  - Trend Lines
  - Moving Average
  - Overlays
  - Indicators
  - Event Annotations
  - Fibonacci Tools
