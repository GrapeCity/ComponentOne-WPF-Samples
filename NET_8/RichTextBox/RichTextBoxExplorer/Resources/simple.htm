Display and edit formatted text as HTML documents with ComponentOne RichTextBoxTM for WinRT XAML. Use the control to display HTML content from the Web or use it as a rich text editor. The C1RichTextBox control supports:

  •  Most text decorations, alignments and styles
  •  Bulleted and numbered lists
  •  Clipboard and document history (undo and redo)
  •  Hyperlinks: ComponentOne website
  •  Insert tables and pictures
